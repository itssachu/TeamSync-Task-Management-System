# 🚀 TeamSync - Task Management System

TeamSync is a modern and responsive **Task & Project Management System** built using **ASP.NET Web Forms**, **C#**, **SQL Server**, **Bootstrap 5**, and **Chart.js**.

The application helps teams manage projects, tasks, users, deadlines, analytics, and productivity in one centralized platform.

## ⚡ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/itssachu/TeamSync-Task-Management-System.git
```

---

### 2️⃣ Open in Visual Studio

Open the solution/project in:

* Visual Studio 2019 or later

---

### 3️⃣ Configure Database

Update your SQL Server connection string inside:

```bash
Web.config
```

Example:

```xml
<connectionStrings>
  <add name="con"
       connectionString="Data Source=SERVER_NAME;Initial Catalog=TeamSyncDB;Integrated Security=True"
       providerName="System.Data.SqlClient"/>
</connectionStrings>
```

---

### 4️⃣ Run the Project

Press:

```bash
F5
```

OR

```bash
Ctrl + F5
```
